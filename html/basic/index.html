    3. The key to a good code is also simplicity. To achieve a certain result, developers may use complex logic but this should be minimised while writing a good code. Simple code does not always depend on the length of the code. The code must also be conceptually simple. An added advantage is if the code is also suitably compact. More code can be viewed on a page and programmer keystrokes can be reduced.
